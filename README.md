# WTHEME-SUBLIME

My setup for VIM text editor. Someone may like the setup but it's here for keep a backup of my setup.

## Preview

![Screenshot](https://i.imgur.com/t16dDw4.png)

## How To:

      git clone https://github.com/wuseman/WVIM
      cd WVIM; mkdir -p ~/.vim/
      cp -r headers ~/.vim/
      cp .vimrc ~/
